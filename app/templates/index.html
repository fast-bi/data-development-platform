Streamline Customer Fast.BI Deployment

Fast.BI Tenant Initialization Wizard is designed to simplify the process of setting up new Customer environment. This comprehensive guide will walk you through the following key stages:

  1. Cloud Provider
  2. Customer values Input
  3. Secrets Generation
  4. Infrastructure Deployment
  5. System Services Deployment
  6. Identity Provider (IDP) Configuration
  7. Data Repository Setup
  8. Data Services Deployment
  9. CI/CD Pipeline Configuration
  10. Deployment Finalization

By following this wizard new Customer tenant will be deployed and integrated into the Fast.BI environment, ensuring all necessary components for Fast.BI Data Platform.
